Arabe > grammaire > verbe > régulier  > forme 1 > kabura (u) - كَبُرَ ـُ = grandir

Les verbes de forme 1 dont la voyelle de l'accompli est u - ـُ ont aussi obligatoirement u - ـُ comme voyelle de l'inaccompli. Ils se conjuguent comme kabura (u) - كَبُرَ ـُ = grandir
Ce sont exclusivement des verbes intransitifs, dont le sens est "être ou devenir ...". Ils correspondent à un adjectif du schème fa3îl -  فاعِيل : ici kabîr - كَبِير = grand.
Autres verbes : être ou devenir ... petit = صَغُرَ ـُ   < صَغِير / vieux = قَدُمَ ـُ < قَدِيم   / proche = قَرُبَ ـُ  < قَرِيب / étrange =  غَرُبَ ـَ  <غَرِيب /  noble =كَرُمَ < كَرِيم  / stupide = سَخُفَ ـُ < سَخِيف
